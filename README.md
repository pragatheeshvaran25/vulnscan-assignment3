# VulnScan - Web Vulnerability Scanner
### Infosys Springboard 6.0 - Assignment 3

## What this project does
Scans websites for security vulnerabilities and sends automatic email alerts.

## Files
- scanner.py - Checks 5 vulnerability types
- dashboard.py - Shows results in web dashboard  
- alerts.py - Sends automatic email alerts

## How to run
1. Install libraries: pip install streamlit plotly requests python-dotenv
2. Add your Gmail details to .env file
3. Run: python -m streamlit run dashboard.py
4. Open browser at http://localhost:8501

## Libraries used
- streamlit, plotly, pandas, requests, python-dotenv
